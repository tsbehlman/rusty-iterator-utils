# rusty-iterator-utils

[![Build Status](https://travis-ci.org/tsbehlman/rusty-iterator-utils.svg?branch=master)](https://travis-ci.org/tsbehlman/slim-cover)

A collection of utility functions for [iterables and iterators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Iteration_protocols) inspired by [Rust's Iterator trait](https://doc.rust-lang.org/std/iter/trait.Iterator.html).
